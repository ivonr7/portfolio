Blob Detection in Python

link to project

For this project I was at BC Cancer. It was an exploratory project seeing if we could characterize if our images were in focus from the radii of Bead images from our microscope. Bead images were little flurophores (small object the emit light if excited by a certain wavelenght) ie little circles in the image that due to physics would get bigger if the object was out of focus or smaller if it was in focus. This project involved increasing contrast using autoscaling. Using adaptive thresholding to convert the image to binary and creating nicer circles using morphological opening and closing. It finally used scikit images region props function to get features. We then filter based on eccentricity and diameter.

Using BERTopic for topic modeling

link to project

For this project I was tasked with getting the main topics for the ShareGPT dataset for validating it's use in a paper extending GPTCache (a cache store for users of chatGPT to decrease costs). For this oroject I tried a couple different approaches strating with the Berkely Neural Parser as well as basic clustering with embedding. I then stumbled upon the BERTopic project which was an extension of the embedding->clustering pipeline. It also added a characterization technique which would search the cluster for words that were highly frequent in a single question but infrequent across all the questions mostly returning nouns which could be characterized as topics. First from the dataset I had to do some cleaning. I found the best performance came when only choosing the first question form each topic. The assumption is that often the first question contains the topic. After running the dataset through the pipeline we got many topics that seemed very valid.
